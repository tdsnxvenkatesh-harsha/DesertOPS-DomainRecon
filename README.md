# Domain Dashboard
This project is a simple frontend help review and analyze a large list of domains across TD SYNNEX and related brands.

## What is Does?
  - Table view of all domains and their status
  - Quick indicators for:
    - Active / inactive
    - Redirects
  - Broken pages
  - Content status (maintained, outdated, abandoned)
  - Risk or purpose tagging (ex: defensive, legacy, typo domain)
  - Filters to sort and explore domains
